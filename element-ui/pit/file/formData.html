# 表单校验有值，但始终不通过

  • 第一种情况：el-form-item标签绑定的prop字段跟v-model绑定的字段不一致。
  • 第二种情况：:model="formData"是否挂载成功，操作的是否是当前表单。
  • 第三种情况：:rules="rulesForm"校验具体规则是否绑定成功并生效，rulesForm建议写在computed中。
  • 第四种情况：初始化formData是否含有v-model绑定的所有字段，如这里的name。
  • 特殊情况：新增与编辑共用表单数据回显易出现校验问题，建议回显赋值前清空一次校验。
    • 清空校验结果：this.$nextTick(()=>this.$refs.ruleForm.clearValidate())
    • 重置表单校验：this.$nextTick(()=>this.$refs.ruleForm.resetFields())
<el-form :model="formData" :rules="rulesForm" ref="ruleForm" class="demo-ruleForm">
    <el-form-item label="姓名" prop="name">
        <el-input v-model="formData.name"></el-input>
	</el-form-item>
	<el-form-item>
        <el-button type="primary" @click="submitForm('ruleForm')">确认</el-button>
	</el-form-item>
</el-form>
